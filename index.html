All models

In this interface model statistics can be sorted and filtered. The value column contains the percentage explained for each component, or the loglikelihood statistic or n respectively.

Abbreviations for the matrices: G: Common genetic. K: Pedigree genetic. F: Family. S: Sibling. C: Couple.

Methods: GREML-KIN: our method including related individuals, adjusting for shared environment. GREML-MS: GREML stratified by minor allele frequency bin.

Other abbreviations: g: general factor of intelligence. logL: log-likelihood for this model. n: sample size for this model. SE: Standard errors for the percentage explained by a component. MAF: Minor allele frequency.
